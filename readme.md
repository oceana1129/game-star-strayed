# Star Strayed

A 2D side-scrolling mobile platformer where players guide Kitty through a space adventure after being separated from its owner. Featuring narrative-driven progression, platforming challenges, collectibles, and progressively introduced mechanics, the game combines exploration, action, and accessibility into a polished mobile experience.

## How It's Made:

**Tech used:** Android Studio, C#

We developed _Star Strayed_ as a mobile Android game focused on approachable platforming and gradual player onboarding. Players progress through ten handcrafted levels, each introducing new mechanics such as double jumping, dashing, moving platforms, falling platforms, checkpoints, hazards, and enemies. Dialogue guides players through each new mechanic while collectible fish encourage exploration beyond the critical path. To create a more immersive experience, we implemented sound effects, background music, haptic feedback, persistent level progression, a health and lives system, checkpoints, and a settings menu that allows players to control audio and resume or restart their adventure.

## Playtesting

Throughout development, we focused on making the game accessible to new players by gradually introducing mechanics instead of overwhelming them all at once. The level progression helped players naturally learn new abilities while narrative dialogue reinforced gameplay concepts. We also emphasized responsive controls, clear feedback, and rewarding exploration to create an engaging experience for players of varying skill levels.

## Optimizations

One of our primary design goals was creating a smooth learning curve. Rather than introducing every mechanic immediately, each level builds upon the previous one by teaching a single new concept before combining mechanics in later stages. We also incorporated checkpoints, persistent save data, audiovisual feedback, and haptic responses to reduce player frustration and improve the overall gameplay experience. Given additional development time, we would expand the number of levels, introduce boss encounters, add additional enemy types and abilities, create more environmental variety, and further develop the game's narrative.

## Lessons Learned:

This project strengthened our understanding of mobile game development, user experience design, and level progression. We gained experience designing engaging platforming mechanics, implementing responsive player controls, managing game state across multiple levels, and balancing challenge through gradual difficulty scaling. Most importantly, we learned how thoughtful onboarding, player feedback, and iterative level design contribute to creating an enjoyable and accessible game experience.
